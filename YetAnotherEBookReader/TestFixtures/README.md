# UI test EPUB fixture

`UI Test Fixture.epub` is a minimal EPUB authored specifically for D.S. Reader
UI tests. Its text, metadata, and stylesheet are original project content; no
third-party book, image, font, or other copyrighted material is included.

The fixture is dedicated to the public domain under [CC0 1.0 Universal](https://creativecommons.org/publicdomain/zero/1.0/)
and may be redistributed with the application. The EPUB metadata also records
the CC0 dedication. It is bundled only as a UI-testing resource and is copied
into the isolated mock book cache when `--ui-testing-mock-library` is present.
